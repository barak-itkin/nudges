# Nudges

An app to remind me to do things on my laptop, without interfering with meetings.

Nudges is a word-play on the Hebrew slang ניג׳ס (nid-jes) - someone that
annoyingly requests/reminds you things, and nudge, which is what this app does
at the end of the day.

## Disclaimers

1.  This app is in early beta/POC stages, don't use for anything critical
2.  While not vibe-code, it's LLM assisted since my MacOS knowledge is minimal.
    There are likely better ways to do some of the things I'm doing here.
